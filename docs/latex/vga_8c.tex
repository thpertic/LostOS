{\footnotesize\ttfamily void update\+\_\+cursor (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Moves the little blinking line under the last pressed char. This sends a command to indicies 14 and 15 in the C\+RT Control Register of the V\+GA controller. These are the high and low bytes of the index that show where the hardware cursor is to be \textquotesingle{}blinking\textquotesingle{}. To learn more, you should look up some V\+GA specific programming documents.
