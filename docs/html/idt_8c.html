void idt_init ( )

Set the IDT in memory.

The Interrupt Descriptor Table (IDT) is specific to the IA-32 architecture. It is the Protected mode counterpart to the Real Mode Interrupt Vector Table (IVT) telling where the Interrupt Service Routines (ISR) are located (one per interrupt vector).

It is similar to the Global Descriptor Table in structure.
