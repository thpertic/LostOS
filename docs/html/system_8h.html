struct idt_gate __attribute__ ( (packed)  )

The size is the size of the table subtracted by 1. The offset is the linear address of the table itself.

The offset is a 32 bit value, split in two parts. It represents the address of the entry point of the ISR. The selector is a 16 bit value and must point to a valid CS descriptor in your GDT. The type_attr is specified here: 7 0 +—+—+—+—+—+—+—+—+ | P | DPL | S | GateType | +—+—+—+—+—+—+—+—+
