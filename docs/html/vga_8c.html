◆ update_cursor()

void update_cursor ( )

Moves the little blinking line under the last pressed char. This sends a command to indicies 14 and 15 in the CRT Control Register of the VGA controller. These are the high and low bytes of the index that show where the hardware cursor is to be 'blinking'. To learn more, you should look up some VGA specific programming documents.
